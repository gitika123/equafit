\section{Sprint Retrospective}

\begin{tcolorbox}[wellbox]
  \begin{itemize}[leftmargin=*, noitemsep]
    \item \textbf{Complete feature delivery:} All 13 development tasks were completed on or before the sprint deadline.
    \item \textbf{Tech stack compatibility:} Next.js App Router, Tailwind, and Framer Motion worked seamlessly, enabling rapid prototyping.
    \item \textbf{User flow cohesion:} The period-aware suggestion system and post-workout feedback loop integrated smoothly with no refactoring needed.
    \item \textbf{Communication cadence:} Regular Zoom and in-class check-ins kept both members fully synchronized throughout the sprint.
  \end{itemize}
\end{tcolorbox}

\vspace{10pt}

\begin{tcolorbox}[issuebox]
  \begin{itemize}[leftmargin=*, noitemsep]
    \item \textbf{Content volume decision (resolved):} Early in the sprint, we debated feasibility of 30-day routines for all 8 goal groups. Resolved by defining a \textit{shared exercise pool} filtered per goal, reducing duplication.
    \item \textbf{Video hosting uncertainty (ongoing):} YouTube embed links were used as a short-term fix. Hosting proprietary video via AWS S3 is an open question for Sprint \#2.
    \item \textbf{No external user testing yet:} All testing was internal. No usability data from real college students has been collected — planned for Sprint \#2.
    \item \textbf{Vibe Sync not integrated:} The mood-based playlist feature was deprioritized to keep focus on core flows; moved to Sprint \#2.
  \end{itemize}
\end{tcolorbox}

\section{Sprint \#2 Planning}

\begin{tcolorbox}[nextsprint]
  \textbf{Process Changes:}
  \begin{itemize}[leftmargin=*, noitemsep, topsep=4pt]
    \item Introduce a \textbf{Definition of Done (DoD)} checklist for each task, including a brief manual test scenario.
    \item Add a \textbf{mid-sprint demo} (after $\sim$1 week) where both members show their progress to each other.
    \item Maintain the 3–4 day meeting cadence that proved effective in Sprint \#1.
  \end{itemize}

  \vspace{6pt}
  \textbf{Sprint \#2 Feature Goals:}
  \begin{itemize}[leftmargin=*, noitemsep, topsep=4pt]
    \item \textbf{Vibe Sync:} Integrate mood-based music playlist API (Spotify or YouTube Music).
    \item \textbf{Video hosting:} Evaluate AWS S3 vs. YouTube embed; implement the chosen strategy.
    \item \textbf{User testing:} Recruit 3–5 college students matching Alex and Maya personas; collect qualitative feedback.
    \item \textbf{Diet Fuel Guide:} Build weekly budget-friendly diet tips section (PRD Feature \#6).
    \item \textbf{Accessibility audit:} Verify high-contrast colors and scalable text per PRD non-functional requirements.
    \item \textbf{Performance testing:} Confirm routine library loads in under 2 seconds on a throttled connection.
  \end{itemize}
\end{tcolorbox}
